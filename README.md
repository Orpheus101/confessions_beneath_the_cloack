# Confessions Beneath the Cloak

An interactive React experience: six letters, six confessions. A poetic journey through silence, longing, and unspoken truth.

## Overview

"Confessions Beneath the Cloak" is a beautifully crafted React component that presents an intimate storytelling experience. Each letter unfolds through interactive envelopes, accompanied by ambient sounds and smooth animations that create an immersive atmosphere.

## Features

- **Six Interactive Letters**: Each letter reveals a piece of a story through elegant card interactions
- **Ambient Soundscapes**: Custom audio synthesis creates unique atmospheres for each letter
- **Smooth Animations**: Grain overlays, particle effects, and fade transitions
- **Responsive Design**: Works seamlessly across different screen sizes
- **Phase-Based Theming**: Color schemes and aesthetics evolve as you progress through the letters
- **Quote Integration**: Each letter features a meaningful quote that appears as you read

## Usage

Import the component into your React application:

```jsx
import ConfessionGame from './confession_game.jsx';

export default function App() {
  return <ConfessionGame />;
}
```

## Technology Stack

- **React 18.2+**: UI framework
- **Web Audio API**: Sound synthesis and ambience
- **CSS3**: Animations and styling
- **SVG**: Graphics and effects

## Browser Support

Works best in modern browsers that support:
- Web Audio API
- CSS Animations & Filters
- CSS Grid
